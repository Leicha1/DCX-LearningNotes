# 1.字节

1. 字节：计算机中存储数据的最小计量单位，用byte或者B表示

2. 存储数据的计量单位换算：
8bit = 1B
1024B = 1KB
1024KB = 1MB
1024MB = 1GB
1024GB = 1TB
